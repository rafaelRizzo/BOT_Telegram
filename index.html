BOT PARA TELEGRAM

Esse site foi com o intuito de ajudar a enviar mensagem usando a API do telegram!

  1. Nenhuma imagem ou texto é enviado diretamente por nós.
  2. Não cobramos nenhum valor para a utilização de nossos serviços, porém, fique a vontade para contribuir com uma doação!
  3. Metade de todas as doações recebidas são utilizadas para ajudar instituições de animais de rua abanadonados!
  4. Qualquer abuso ou spam é de total responsabilidade do usuário!
  5. Também é responsabilidade do dono do BOT e do CHAT ID fornecer esses dados a um terceiro para ser utilizado em nosso site.
  6. Não cabe a nós o envio de imagens ou mensagens inapropriadas.
  7. Em nenhuma hipótese informe as credencias do seu BOT e do CHAT ID para alguem não confiável.
  8. Não entraremos em contato sem que você abra um atendimento de ajuda conosco, nunca iremos pedir senhas ou dados pessoais, cuidadao com golpes!
  9. Alguns termos podem sofrer alterações, serem removidos, adicionados ou reavaliados de acordo com o tempo não determinado por isso fique sempre atento!
  10. É obrigatório aceitar com os termos a cima para utilização do nosso serviço!
